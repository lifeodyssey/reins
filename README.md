# Reins

Harness engineering framework for Claude Code.

## Install as plugin

```bash
claude plugin marketplace add lifeodyssey/reins
claude plugin install reins
```

## Install as standalone binary

Download from [Releases](https://github.com/lifeodyssey/reins/releases):

| Platform | Binary |
|----------|--------|
| macOS (Apple Silicon) | `reins-macos-arm64` |
| macOS (Intel) | `reins-macos-x64` |
| Linux (x64) | `reins-linux-x64` |
| Windows (x64) | `reins-windows-x64.exe` |

```bash
chmod +x reins-macos-arm64
./reins-macos-arm64
# Opens http://localhost:7860
```

Requires: [Claude Code CLI](https://claude.ai/code) installed and authenticated.
